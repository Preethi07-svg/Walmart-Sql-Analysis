# Walmart-Sql-Analysis
End-to-End SQL project analyzing Walmart Sales data using MySQL with data cleaning, feature engineering, and business insights.

## Project Overview

This project analyzes Walmart Sales data using MySQL. The project demonstrates SQL skills including data cleaning, feature engineering, exploratory data analysis, and business analysis by solving real-world business questions.

---

## Objectives

- Analyze Walmart sales performance
- Perform data cleaning
- Create new features for better analysis
- Answer business-related questions using SQL
- Generate meaningful business insights

---

The dataset contains sales transactions from Walmart stores.
